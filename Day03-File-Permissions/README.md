# Day 03 - File Permissions & Ownership

## Topics Learned

- Read Permission (r)
- Write Permission (w)
- Execute Permission (x)
- chmod
- chown
- Ownership
- 777
- 755
- 644

## Interview Questions

### What is chmod?
Used to change permissions.

### What is chown?
Used to change ownership.

### Difference between chmod and chown?

chmod changes permissions.
chown changes owner.

### What is 777?

Read, Write, Execute for everyone.

### What is 755?

Owner: Read, Write, Execute
Group: Read, Execute
Others: Read, Execute

### What is 644?

Owner: Read, Write
Group: Read
Others: Read
